# Keyboard Shortcuts

| Action | macOS | Windows / Linux |
|---|---|---|
| **Undo** | `Cmd+Z` | `Ctrl+Z` |
| **Redo** | `Cmd+Shift+Z` | `Ctrl+Y` |

Additional shortcuts may be added in future releases.

---

## Tips

- Undo/Redo history is **per sheet** — undoing on one sheet does not affect another.
- The undo stack is preserved as long as your browser session is active. Refreshing the page clears the history (but your last saved snapshot is unaffected).
