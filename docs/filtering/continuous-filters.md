# Continuous Filters

Continuous filters let you restrict a numeric measure or a numeric dimension to a specific range.

---

## Adding a continuous filter

Drag any **Measure** or continuous numeric field onto the **Filters** drop zone.  
A range slider opens showing the minimum and maximum values found in the data.

---

## Setting a range

- Drag the **left handle** to set the minimum value.
- Drag the **right handle** to set the maximum value.
- Or type exact numbers directly into the **min / max input boxes** below the slider.

Click **Apply** to run the query with the new range applied.

---

## Full range (no filter)

Set the slider back to the full min–max extent to effectively remove the range restriction without deleting the filter card.

---

## Removing a continuous filter

Click the **×** on the filter pill in the filter bar.
