Entry & Routing
index.tsx
React entry point
Mounts provider tree: DataSourceProvider → VisualizationProvider → ConnectionProvider → App.
App.tsx
BrowserRouter, SheetProvider
Tab management, save/load menu, snapshot gallery. Routes: / → DataSourceSelectionPage, /visualize → VisualizationPage.
State Management
React Contexts (shared across app)
DataSourceContext
contexts/DataSourceContext
DB/table/field selection, virtual columns, session filters, Hive partitions. Shared across all sheets.
ConnectionContext
contexts/ConnectionContext
Connection lifecycle: connect, disconnect, connection details. Wraps API calls.
SheetContext
contexts/SheetContext
Multi-sheet workspace. Active sheet, CRUD on sheets, sheet switching.
VisualizationContext
contexts/VisualizationContext
Per-sheet viz state: axes, filters, encodings, query result, loading, chart type. Remounts on sheet switch.
UndoRedoContext
contexts/UndoRedoContext
Per-sheet undo/redo stacks. Tracks visualization state changes.
Zustand Stores (external state)
sheetRenderCacheStore
stores/sheetRenderCacheStore
Caches queryResult + chartSpec per sheet for fast tab switching without re-querying.
selectionStore
stores/selectionStore
Multi-select for fields: shift-click, range select in the fields panel.
Pages & Layout
DataSourceSelectionPage
pages/DataSourceSelectionPage
Connection selection. Shows connection forms (CSV, ClickHouse, Kaggle, Hive Parquet), connect/disconnect, config load.
VisualizationPage
pages/VisualizationPage
Main workspace. Wraps per-sheet VisualizationProvider + UndoRedoProvider. Contains resizable panel layout.
AppLayout
components/Layout/
Resizable panels via react-resizable-panels. PanelResizeHandleWithToggle, CollapsedPanelStrip.
UI Components
Connection Forms
CsvConnectionForm
File upload for CSV/Parquet. Multipart upload, file validation.
ClickHouseConnectionForm
Host, port, database, user/password form.
KaggleConnectionForm
Kaggle dataset search and import.
HiveParquetConnectionForm
Two-phase: path entry → partition selection.
Visualization Panels

Left sidebar: field search, table list, field categories, drag-to-axis. FieldChip, CompactMetadataSelector.
ChartArea

FieldOverridesPanel
Chart type, color field, size field, encoding overrides per field.
ColorPanel
Color encoding: ColorDropZone, ColorSchemeSelector, ManualColorSelector.
LegendPanel
Chart legend. LegendStack, BackgroundLegendPanel.
TableView
AG Grid data table with react-window lazy loading.
MeasureGroupsPanel
Measure groups management for multi-measure charts.
Chart Rendering
ChartRenderer
Renders Observable Plot output. DOM mounting, resizing, accessibility.
MultiPlotGrid
Faceted/multi-plot grid layout. PlotArea, BrushOverlay, XAxes, YAxes, FacetLabels.
Other Components
VirtualColumnManager
Create/edit virtual columns and binning. BinConfigDialog.
SaveLoadMenu
Export/import configs, snapshot gallery, connection restore.
DateTimeRangeFilter
DateTime range picker, DateTimePartMenu for date part extraction.
FieldChip
Draggable field tag. FieldChipLabel, FieldContextMenu, FieldAliasDialog.
Custom Hooks
useVisualizationState
Aggregates DataSource + Visualization + metadata into a single unified state.
useDragDrop
Axis/filter/color/size drag-and-drop. Field assignment to encoding channels.
useQueryExecution
Coordinates query build → execution → result. Triggers on viz state changes.
useQueryBuilder
Builds QueryDescription from axes, filters, encodings, virtual columns.
useMetadataOperations
Fetch databases, tables, columns from backend. Populates DataSourceContext.
useGlobalFilters
Session-level vs sheet-level filter management.
useUndoRedo
Undo/redo hook. Tracks viz state snapshots per sheet.
useConnectionForm
Connection form state management for all connection types.
useRenderingCoordinator
Tracks plot rendering lifecycle for loading states.
useSheetRenderCache
Sync data source version for cache invalidation on sheet switch.
Services & API
API Layer (services/api/)
apiClient
Base HTTP client. API_BASE URL, AbortController for cancellation, X-Tab-Id header.
connectionApi
connect, disconnect, connectHive, loadPartition, addFiles.
metadataApi
listDatabases, listTables, listColumns, getFieldStats, getDistinctValues.
queryApi
executeQuery (JSON), executeQueryArrow (Arrow IPC), executeQueryArrowRaw.
snapshotApi
listSnapshots, saveSnapshot, loadSnapshot, deleteSnapshot.
kaggleApi
searchKaggleDatasets, listKaggleFiles.
Query & Chart Generation
queryBuilder/
Converts visualization state (axes, filters, measures) into QueryDescription for the backend.
observable-plot-generator/
Chart generation on @observablehq/plot. Chart types (line, bar, scatter, cell, gantt, tick), faceting, grid, colors, tooltips.
optimizationHintGenerator
Generates optimization hints for backend query optimizer based on field types and query structure.
configurationService
Export/import full app config. Validation, connection reconstruction, snapshot serialization.
Local Execution (DuckDB WASM)
queryExecutionOrchestrator
Orchestrates local (DuckDB WASM) vs remote (backend) query execution.
queryDecisionEngine
Decides local vs remote based on data size, query complexity, caching.
localSqlBuilder
Builds SQL for local DuckDB WASM execution.
duckdbService
DuckDB WASM for local caching and query. In-browser SQL engine.
arrowResultAdapter
Converts Apache Arrow IPC results to row-based data for rendering.
Shared Infrastructure
types/
TypeScript types
Field, QueryDescription, QueryResult, Filter, Connection, Sheet, VirtualColumn, SavedConfig, Kaggle, Tooltip, etc.
utils/
Helpers
fieldUtils, filterActions, syntheticFields, sheetConfigHash, binningUtils, scaleInversion, dragDataStore, tabSession.
config/
App configuration
colorSchemes (categorical, sequential, diverging), loadingConfig (modal timeouts).

Query flow: UI Panel → useQueryBuilder → QueryDescription → queryApi.executeQuery → Backend → QueryResult → observablePlotGenerator → ChartRenderer

Local path: useQueryExecution → queryExecutionOrchestrator → queryDecisionEngine → localSqlBuilder → DuckDB WASM → arrowResultAdapter

Connect flow: ConnectionForm → connectionApi.connect → Backend → DataSourceContext.setTables → metadataApi → FieldsPanel
